# ccgonowdep

A cross-platform C++ library built with CCGO.

## Description

ccgonowdep library - a dependency library for ccgonow.

## Platforms

- Android (armeabi-v7a, arm64-v8a, x86_64)
- iOS
- macOS
- tvOS
- watchOS
- OpenHarmony/HarmonyOS (armeabi-v7a, arm64-v8a, x86_64)
- Windows
- Linux

## Installation

### Android (Maven)

```gradle
implementation 'com.mojeter.ccgo.samples:ccgonowdep:1.0.0'
```

### iOS/macOS (CocoaPods)

```ruby
pod 'ccgonowdep', '~> 1.0'
```

### OpenHarmony (OHPM)

```bash
ohpm install ccgonowdep
```
