# CCGONOWDEP

Welcome to CCGONOWDEP documentation.

## Overview

CCGONOWDEP is a cross-platform C++ library built with [CCGO](https://github.com/ccgo-group/ccgo).

## Features

- Cross-platform support (Android, iOS, macOS, Windows, Linux, OpenHarmony)
- Modern C++ implementation
- Easy integration with native platforms

## Quick Start

```cpp
#include <ccgonowdep/api/native/ccgonowdep.h>

int main() {
    // Your code here
    return 0;
}
```
