# ccgonow

A cross-platform C++ library built with CCGO.

## Description

ccgonow library - demonstrates cross-platform C++ development with CCGO build system.

## Platforms

- Android (armeabi-v7a, arm64-v8a, x86_64)
- iOS
- macOS
- tvOS
- watchOS
- OpenHarmony/HarmonyOS (armeabi-v7a, arm64-v8a, x86_64)
- Windows
- Linux

## Installation

### Android (Maven)

```gradle
implementation 'com.mojeter.ccgo.samples:ccgonow:1.0.2'
```

### iOS/macOS (CocoaPods)

```ruby
pod 'ccgonow', '~> 1.0'
```

### OpenHarmony (OHPM)

```bash
ohpm install ccgonow
```

## Dependencies

- ccgonowdep
